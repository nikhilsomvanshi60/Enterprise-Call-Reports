🔑 Recover Passcode

Agar aap security passcode bhool gaye hain, toh ise niche diye gaye tariko se recover/reset kar sakte hain:

  1. PC Dashboard se: Apne computer par PC Dashboard open karein aur top right header me 🔑 Change PIN par click karke reset karein.
  2. Manual Reset (Server File): Host computer par project folder me jaakar data/security_config.json file ko notepad me open karein. "pin": "..." ki value me apna naya plaintext PIN likh kar file save karein aur server restart karein (startup par ye auto-hash ho jayega).
